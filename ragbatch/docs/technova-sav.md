# TechNova — Service après-vente

## Politique de retour

Les clients peuvent retourner tout produit sous 45 jours avec le code
RETOUR-45. Les frais de retour sont offerts à partir de 50 € d'achat.
En dessous de ce montant, les frais de retour s'élèvent à 4,90 €.

Le remboursement est effectué sous 5 jours ouvrés après réception du
colis en entrepôt, sur le moyen de paiement d'origine.

## Support technique

Le support technique TechNova est ouvert du lundi au samedi, de 8h à 19h.
Contact prioritaire pour les pannes bloquantes : urgence@technova.example
(réponse sous 2 heures). Pour les demandes non urgentes, le délai de
réponse moyen est de 24 heures via support@technova.example.

## Garantie

La garantie constructeur TechNova dure 3 ans sur les produits de la
gamme Pro et 1 an sur la gamme Essentielle. L'extension de garantie
PremiumCare ajoute 2 ans supplémentaires, pour 49 € par produit.
La garantie couvre les défauts de fabrication, pas la casse accidentelle.
